## 第二层：量能确认

**这是区分胜者和败者最关键的指标。历史数据：胜者组 post/pre=1.32x，败者组 post/pre=0.69x。**

### 量比计算方法

```
        信号触发时刻
             ↓
  ┌─────┬─────┬─────┬──┬──┬──┬──┬──┐
  │ -10 │ -5  │  0  │+5│10│15│20│25│  ← 相对信号的时间(分钟)
  ├─────┼─────┼─────┼──┼──┼──┼──┼──┤
  │ K-2 │ K-1 │Ksig │K1│K2│K3│K4│K5│  ← 5分钟K线
  └─────┴─────┴─────┴──┴──┴──┴──┴──┘
         ← pre →        ← post →

pre  = 信号前5根K线(K-4~K0)的平均成交量
post = 信号后5根K线(K+1~K+5)的平均成交量
量比 = post / pre
```

### 判断标准

| 量比 | 含义 | 操作 | 仓位 |
|------|------|------|------|
| **> 1.2** | 放量，真实买盘进场 | ✅ 通过 | 正常仓位 |
| **0.8 ~ 1.2** | 量能持平，方向不明 | ⚠️ 降级通过 | 减半仓位(5%) |
| **< 0.8** | 缩量，死猫反弹 | ❌ **直接丢弃** | 0 |

> 量比<0.8时，75%的概率是pump_crash路径（先拉高再崩盘）。量比>1.2时，60%的概率是V_reversal路径（先跌后涨）。

## 第四层：仓位调整

### 基础仓位

```
单笔基础仓位: 总资金的 8%
最大并发持仓: 5个
日最大亏损:   总资金的 15%
```

### 风险降权规则

| 条件 | 降权幅度 | 原因 |
|------|---------|------|
| 年龄<24h 且 流动性<30K | **-50%** | 新币洗盘深度深 |
| pool/mcap > 50% | **-30%** | 项目方锁流动性，易诱多 |
| 年龄在 7天~30天 | **-30%** | 沉寂期，平均最大收益仅+26% |
| 流动性 < 30K | **-50%** | 池子太浅 |
| 因子评分 50-65 | **-40%** | 信号质量中等 |
| 因子评分 < 50 | **-70%** | 信号质量低 |

### 仓位计算公式

```
最终仓位% = 8% × (1 - 降权1) × (1 - 降权2) × ...
最终仓位% = clamp(最终仓位%, 1%, 15%)
持仓金额   = 总资金 × 最终仓位%
```

## 第五层：入场后风控

### 止损

```
硬止损价 = 加权入场均价 × (1 + 止损%)

策略A: 入场价 × 0.78  (-22%)
策略B: 入场价 × 0.85  (-15%)
策略C: 入场价 × 0.80  (-20%)
策略D: 入场价 × 0.88  (-12%)
策略E: 入场价 × 0.82  (-18%)
```

### 移动止盈

```
激活条件: 价格上涨 ≥ +50%（从入场价）
激活后:   止盈价 = 历史最高价 × 0.85（从最高点回落15%卖出）
保本保护: 到过+50%以上后跌回入场价±5% → 无条件平仓
```

### 分批止盈（以策略A为例）

```
入场价 = $0.0001000

涨到 +30% ($0.0001300):
  → 卖出 50% 仓位，锁定 30% 利润

涨到 +60% ($0.0001600):
  → 卖出 30% 仓位，锁定 60% 利润

涨到 +100% ($0.0002000):
  → 卖出剩余 20%，锁定 100% 利润

如果涨到 +150% 后回落:
  → 移动止盈在 +135%（高点×0.85）
```

### 日风控

```
单日亏损达总资金 15% → 当日停止一切交易
已有持仓在亏损中     → 不新增仓位
```

## 执行检查清单

收到信号时，逐项打勾：

```
□ 1. signal_type 是 abnormal / new_revival / quiet_runup？
     (drop_40w/50w → 直接丢弃)

□ 2. current_mcap 在 $30K ~ $5M？
     (<30K或>5M → 直接丢弃)

□ 3. liquidity > $10,000？
     (<10K → 直接丢弃)

□ 4. price_change_pct 在 15% ~ 200%？
     (<15% → 无动能; >200% → 追高陷阱)

□ 5. ATH比 避开 5%~10% 半山腰区间？

□ 6. 等待15-25分钟后:
     post/pre 量比 > 1.2？
     (<0.8 → 死猫反弹，丢弃; 0.8~1.2 → 减半仓位)

□ 7. 匹配到策略A/B/C/D/E之一？

□ 8. 按策略设置止损价（成交后立即设）

□ 9. 按策略设置分批止盈价

□ 10. 移动止盈激活设为入场价×1.5

□ 11. 确认仓位 ≤ 10%总资金

□ 12. 确认并发持仓 ≤ 5个
```
