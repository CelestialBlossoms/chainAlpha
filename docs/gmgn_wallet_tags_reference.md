# GMGN 钱包标签持仓指标参考手册

> 数据来源: `gmgn-cli token info` → `wallet_tags_stat` + `stat` 字段
> 适用链: Solana / BSC / Base / ETH

---

## 一、标签对应的 GMGN 字段

| 标签 | GMGN 字段 | 值类型 |
|------|----------|--------|
| 🤖 高频交易(机器人) | `stat.bot_degen_count` / `bot_degen_rate` | 数量 / 占比 |
| 🐣 新钱包 | `stat.fresh_wallet_rate` / `wallet_tags_stat.fresh_wallets` | 占比 / 数量 |
| 🏄 老钱包 | 无直接字段，需根据 `created_at` 推断 | — |
| 💩 垃圾钱包(老鼠仓) | `stat.top_rat_trader_percentage` / `wallet_tags_stat.rat_trader_wallets` | 占比 / 数量 |
| 💎 蓝筹头部赢家 | `stat.bluechip_owner_count` / `bluechip_owner_percentage` | 数量 / 占比 |
| 💰 聪明钱 | `wallet_tags_stat.smart_wallets` | 数量 |
| 🏭 做市/捆绑钱包 | `stat.top_bundler_trader_percentage` / `wallet_tags_stat.bundler_wallets` | 占比 / 数量 |
| 🏦 CEX 钱包 | `token holders` 中 `addr_type: 2` | 数量 |
| 👑 KOL/知名钱包 | `wallet_tags_stat.renowned_wallets` | 数量 |
| 🎯 狙击手 | `wallet_tags_stat.sniper_wallets` | 数量 |
| 🐋 巨鲸 | `wallet_tags_stat.whale_wallets` | 数量 |
| 🏆 顶级钱包 | `wallet_tags_stat.top_wallets` | 数量 |

## 二、各指标详解

### 🤖 高频交易实时持仓 (Bot Degen)

**含义**: 使用交易机器人（Photon/BullX/Trojan/GMGN 等）高频买卖的钱包。特征是单钱包交易笔数极高、持有时长极短。

**解读**:
- **过高 (>40%)**: 说明盘面由机器人主导，拉升后会有大量程序化卖单，容易出现回调/砸盘
- **过低 (<10%)**: 盘子冷清，没有交易量
- **金狗早期**: 可以偏高（20-40%），因为 bots 是第一批入场者

| 区间 | 判断 |
|------|------|
| 0-10% | 冷清 |
| 10-30% | 正常 |
| 30-40% | ⚠️ 偏高，注意回调 |
| >40% | 🚫 机器人主导，高抛压风险 |

**GMGN 字段**: `stat.bot_degen_count` (数量), `stat.bot_degen_rate` (占比 0-1)

---

### 🐣 新钱包实时持仓 (Fresh Wallet)

**含义**: 近期创建、交易历史短的新钱包。可能是阴谋盘（项目方/做市团队的分发钱包）也可能是真实新用户。

**解读**:
- **过低 (<30%)**: 没有新资金进入，盘子可能已死
- **过高 (>70%)**: 大量新钱包可能是庄家分发筹码制造的假象，容易断头砸盘
- **最佳区间 30-70%**: 既有庄家铺垫又有真实用户

| 区间 | 判断 |
|------|------|
| 0-30% | ❌ 无新资金，冷清 |
| 30-50% | ✅ 健康 |
| 50-70% | ⚠️ 偏高，观察 |
| >70% | 🚫 过度集中，可疑 |

**GMGN 字段**: `stat.fresh_wallet_rate` (0-1)

---

### 🏄 老钱包实时持仓 (OG Wallet)

**含义**: 创建时间早、有长期交易历史的钱包。通常是经历过多个周期的玩家，持仓更稳定不容易被洗下车。

**解读**:
- **越高越好**: 老钱包占比高说明盘子质量高，真正的金狗应该老钱包占大比例
- **过低**: 全是新钱包，可能是短期盘子

| 区间 | 判断 |
|------|------|
| 0-20% | ❌ 质量低 |
| 20-50% | ✅ 正常 |
| 50-70% | ✅ 优质金狗 |
| >70% | 🏆 顶级 |

**GMGN 字段**: 无直接字段，需根据持仓钱包的 `created_at` 自行统计

---

### 💩 垃圾钱包实时持仓 (Rat Trader)

**含义**: 内幕交易/老鼠仓钱包。这些钱包在代币开盘瞬间买入，属于项目方关联地址或提前知道消息的 insider。

**解读**:
- **>5%**: 盘子质量差，内幕交易严重
- **>10%**: 典型的阴谋盘/诈骗盘，随时可能断头
- **0-2%**: 健康

| 区间 | 判断 |
|------|------|
| 0-2% | ✅ 干净 |
| 2-5% | ⚠️ 有嫌疑 |
| 5-10% | 🚫 垃圾盘 |
| >10% | 🚫 纯粹的诈骗盘 |

**GMGN 字段**: `stat.top_rat_trader_percentage` (0-1)

---

### 💎 蓝筹头部赢家实时持仓 (Bluechip Owner)

**含义**: 持有过蓝筹代币（如 BONK/WIF 等）且获利丰厚的钱包。是经过市场验证的"真大佬"。

**解读**:
- **越高越好**: 蓝筹赢家愿意持有说明他们看好这个盘子
- **金狗通常 >15%**: 真正的金狗蓝筹占比不会低
- **<5%**: 没有大资金关注

| 区间 | 判断 |
|------|------|
| 0-5% | ❌ 无大资金 |
| 5-15% | ✅ 正常 |
| 15-30% | ✅ 优质 |
| 30-50% | 🏆 顶级金狗 |

**GMGN 字段**: `stat.bluechip_owner_percentage` (0-1)

---

### 💰 聪明钱实时持仓 (Smart Money)

**含义**: GMGN 算法标记的历史胜率高、盈利稳定的钱包。是链上最值得关注的一批玩家。

**解读**:
- **越高越好**: 聪明钱愿意入手说明基本面/技术面都过关
- **0 个**: 说明聪明钱不碰，需要谨慎
- **>10 个**: 非常积极的信号

| 区间 | 判断 |
|------|------|
| 0 | ⚠️ 无人问津 |
| 1-5 | ✅ 有资金关注 |
| 5-20 | ✅ 强信号 |
| 20-50 | 🏆 聪明钱重仓 |

**GMGN 字段**: `wallet_tags_stat.smart_wallets` (数量)

---

### 🏭 做市/捆绑钱包实时持仓 (Bundler)

**含义**: 使用捆绑交易（bundle）技术的钱包，通常是做市团队或机器人。他们可以在一个区块内打包多笔交易同时执行。

**解读**:
- **出现做市钱包**: 往往是大波拉升的开始，做市团队介入控盘
- **过高 (>30%)**: 庄家控盘严重，散户容易成为 exit liquidity
- **0-5%**: 正常，无做市痕迹

| 区间 | 判断 |
|------|------|
| 0-5% | ✅ 正常 |
| 5-20% | ⚠️ 做市介入 |
| 20-30% | ⚠️ 庄家控盘 |
| >30% | 🚫 高度控盘 |

**GMGN 字段**: `stat.top_bundler_trader_percentage` (0-1)

---

### 🏦 CEX 实时持仓 (Exchange Wallet)

**含义**: 交易所（Binance/OKX/Bybit 等）的钱包持有该代币。

**解读**:
- **CEX 开始持仓**: 往往意味着该 MEME 即将上线该交易所的现货交易对
- **>5%**: 多个交易所囤货，上币预期极强

| 区间 | 判断 |
|------|------|
| 0% | 正常，未到上币阶段 |
| 0-5% | 1 家 CEX 在囤货 |
| 5-10% | 🚀 多 CEX 囤货，上币预期强 |

**GMGN 字段**: `token holders` 中 `addr_type: 2` 且 `exchange` 为 CEX 名称的钱包

## 三、快速评分卡

| 指标 | 满分条件 | 扣分条件 |
|------|---------|---------|
| 机器人占比 | ≤30% | >40% 扣 20 分 |
| 新钱包占比 | 30-50% | <20% 或 >70% 扣 15 分 |
| 老鼠仓占比 | ≤2% | >5% 扣 25 分 |
| 蓝筹占比 | ≥15% | <5% 扣 15 分 |
| 聪明钱数量 | ≥5 | 0 扣 20 分 |
| 捆绑占比 | ≤10% | >20% 扣 10 分 |

**总分 80+**: 🟢 优质
**总分 50-79**: 🟡 一般  
**总分 <50**: 🔴 垃圾

## 四、典型金狗画像

```
🤖 机器人 15-30%     ← bot 和 sniper 最先入场
🐣 新钱包 40-60%     ← 散户 FOMO 追涨
🏄 老钱包 20-40%     ← OG 玩家加持
💩 老鼠仓 <2%        ← 没有 insider rug 风险
💎 蓝筹 15-30%       ← 大佬们都在
💰 聪明钱 5-20 个   ← 聪明钱确认
🏭 捆绑 5-15%        ← 做市团队介入
```
