## 二、三层过滤系统

311 个信号 → 过滤 287 个 → 入场 24 笔（入场率 7.7%）

### 第一层：信号类型过滤（淘汰 62%）

| 信号类型 | 行为 | 决策 | 数据依据 |
|----------|------|------|---------|
| 先涨后崩 / 直接上涨 | 信号后直接拉盘，然后暴跌 | **跳过** | 93% 当前亏损 |
| 先跌后涨 / 回撤后上涨 | 信号后先回撤，再真正拉升 | **继续观察** | 76% 当前盈利 |

### 第二层：回撤深度过滤（淘汰 21%）

| 回撤幅度 | 决策 | 逻辑 |
|----------|------|------|
| 未到 -40% | **不入场** | 没有足够安全垫，宁可错过 |
| 达到 -40% | **挂限价单** | 信号价 × 0.6 入场，天然折扣 40% |

### 第三层：归零检测过滤（淘汰 11%）

| 1 分钟内表现 | 决策 | 数据依据 |
|-------------|------|---------|
| 跌破 -65% | **不买** | 回撤 > 65% 后继续跌的，100% 最终亏损 |
| 稳住 -40%~-65% | **确认入场** | 真正的洗盘到此为止 |

## 三、入场后四道保险

### 保险 1：硬止损（-25%）

```
入场价 × 0.75 = 信号价 × 0.45

触发即割，不犹豫。
```

从 -40% 入场算，-25% 止损对应信号价的 -55%。真正的洗盘行为不会跌穿这个位置，跌穿了就是归零。

### 保险 2：提前砍（-20%）

```
入场后一直没到 +50%，反而跌到 -20% → 主动砍
```

防止慢性失血。不是归零，但也没戏，趁早止损。

### 保险 3：移动止盈（激活条件：+50%）

```
涨到 +50% → 激活 -15% 移动止盈

例：
  涨到 100%  → 回撤 15% 到 85% → 卖出（锁住 85%）
  涨到 500%  → 回撤 15% 到 425% → 卖出（锁住 425%）
  涨到 2000% → 回撤 15% 到 1700% → 卖出（锁住 1700%）
```

| 止盈方式 | 收益率 | 对比 |
|----------|--------|------|
| +50% 全出 | +4.7% | 砍掉了所有大肉 |
| +100% 全出 | +24.5% | 砍掉了 10 倍票 |
| 移动止盈 -15% | **+109.9%** | 让利润奔跑 |

**核心原理**：大肉票的涨幅不是 100%，是 1000%+。固定止盈会砍掉 99% 的利润。移动止盈让赢家充分奔跑，只在大趋势反转时退出。

### 保险 4：保本出

```
到过 +50% → 跌回成本线 ±5% → 保本出

绝不被套后还亏着出。
```

## 七、执行清单

按以下步骤操作，每笔交易前自检：

```
□ 1. 信号是"先跌后涨"/"回撤后上涨"型？      → 不是就跳过
□ 2. 价格已回撤到信号价的 -40% 以下？        → 没到就等
□ 3. 等待 1 分钟，价格稳在 -40%~-65%？       → 跌破 -65% 就跳过
□ 4. 挂限价单入场（信号价 × 0.6 附近）       → 成交
□ 5. 设 -25% 硬止损                          → 不撤不改
□ 6. 等 +50% 触发移动止盈                     → 没触发前如果跌到 -20% 主动砍
□ 7. 移动止盈激活后，回撤 15% 自动卖           → 不手动干预
□ 8. 如果跌回成本线，保本出                    → 不亏就是赚
```

## 八、交易时段

> 基于 401 个信号（含多日 Alpha + Bottom 数据）的时段分析。

### 时段评级

| 交易时段 | 信号数 | 可用率 | 入场 | 胜率 | 均 PnL | 评级 |
|----------|--------|--------|------|------|--------|------|
| **03-07 凌晨黄金段** | 93 | 39% | 10 | 90% | +295% | **S** |
| **12-15 下午** | 78 | 30% | 7 | 86% | +297% | **S** |
| **18-22 晚间黄金段** | 70 | 44% | 7 | 100% | +249% | **S** |
| 08-11 上午 | 61 | 39% | 5 | 80% | +161% | A |
| 23-02 深夜 | 53 | 40% | 1 | 100% | +105% | C |
| **16-17 垃圾时间** | 46 | 26% | **0** | — | — | **避开** |

### 两个黄金窗口

**03:00-07:00 凌晨（亚洲盘前埋伏期）：**
- 美国刚收市、亚洲还没开，流动性薄，容易砸出深坑
- 庄家喜欢在这个时段洗盘吸筹，回撤最深、信号最真
- 入场频率最高（10 笔），90% 胜率

**18:00-22:00 晚间（欧美盘活跃期）：**
- 欧美交易员上线，资金涌入，拉升最有持续性
- 100% 胜率，峰值涨幅最大的票集中在这个时段
- 肉最厚但出现频率稍低（7 笔）

### 垃圾时间

**16:00-17:00：0 笔入场。** 连续 4 天的数据中，这个时段没有任何信号满足入场条件——要么冲高假突破，要么回撤不够深。强制休息。

**23:00-02:00 深夜：** 只有 1 笔入场，信号虽多但大部分是假信号，流动性太薄容易被操纵。

### 每日操作节奏

```
03:00-07:00  ← S: 黄金窗口1，集中盯盘，主力作战
07:00-08:00  ← 收尾、复盘
08:00-11:00  ← A: 辅助窗口，有信号就做，没有不强求
11:00-13:00  ← 午休，市场清淡
13:00-15:00  ← S: 辅助窗口，欧美早盘前夕
15:00-16:00  ← 准备切换
16:00-17:00  ← X: 垃圾时间，强制休息
17:00-18:00  ← 热身
18:00-22:00  ← S: 黄金窗口2，集中盯盘，主力作战
22:00-03:00  ← 休息或轻仓观察
```

### 时段规则

| 规则 | 说明 |
|------|------|
| S 时段全力出击 | 两个黄金窗口内，信号质量最高，优先分配资金和注意力 |
| A 时段选择性交易 | 有信号就做，没有不强求，降低仓位 |
| C 时段轻仓观察 | 深夜信号少碰，假突破多 |
| X 时段强制休息 | 16-17 点不交易，保护资本也保护精力 |
