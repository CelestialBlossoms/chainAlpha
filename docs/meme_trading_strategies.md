# MEME 代币筹码峰交易策略手册

> 基于 GMGN 筹码分布数据的实战策略集合
> 核心原则：一切都是概率，盈亏比高即可，没有绝对盈利模式

---

## 一、前置知识：GMGN 筹码峰数据获取

### 关键 API

```bash
# 获取筹码分布（前100持仓 + 历史快照）
gmgn-cli token holders --chain sol --address <CA> --limit 100

# 历史筹码快照（底部监控表）
SELECT * FROM bottom_top100_snapshots WHERE address = '<CA>' ORDER BY snapshot_ts DESC
```

### 核心指标

| 指标 | GMGN 字段 | 含义 |
|------|----------|------|
| 持仓占比 | `amount_percentage` | 该钱包占总供应百分比 |
| Top10 集中度 | `top_10_holder_rate` | 前10钱包持仓占比 |
| 筹码卖出进度 | `sell_amount_percentage` | 该钱包已卖出买入量的比例 |
| 净流入 | `netflow_usd` | 买入 - 卖出 (USD) |
| 持仓成本 | `avg_cost` | 平均买入价 |

## 二、筹码峰交易必备形态知识

### 2.1 好的筹码峰及核心特征

**原理**: 价格处于低位区间时，主力不再打压价格，通过反复震荡换手，把市场筹码慢慢收集到自己手中，在低位形成一个**价格集中、结构收敛、密度很高**的筹码峰。本质代表：**主力的核心成本区已经建立完成**。

**判断条件**:
1. 价格处于相对低位，且不再下跌，形成明显的价格区间
2. 筹码峰变窄，聚集在低位价格附近
3. 大量筹码卡在同一低价区 → 通常为主力的核心吸筹价格

### 2.2 低位筹码峰的形成

如果早期有多个筹码峰（高位多峰），中后期合并成低位一个大单峰，则说明**主力洗盘完毕，要主升浪**。

**低位筹码峰集中的优势**:
- 筹码越窄说明主力筹码高度一致
- 扛单能力强，不容易出现大波动踩踏
- 爆发时走势很干净

### 2.3 筹码健康上移

筹码峰从低位缓慢往上移动说明：
- 主力持仓成本上移
- 有带筹码往上走的意图
- 整体趋势健康，不是瞬间跳到高位（那是出货筹码峰）

### 2.4 上涨过程筹码不散

上涨时筹码峰不散、不乱跑、筹码依然集中、没有大量抛压，说明**主力还不愿意卖，有继续拉升的潜力**。

## 三、交易策略 1：斐波那契回撤 + 筹码峰入场

### 适用场景

第一波拉升后，高位筹码被洗掉，价格回落，但真正的主力并没有走，而是在低位重新回收筹码。该策略主要适用于**跟随主力低位回收筹码入场**。

### 原理

- 斐波那契回撤用于定位"价格回调深度"
- 筹码峰的下边缘用于确认"主力真实成本防守线"
- 二者结合 = meme 胜率最高、风险最低的回调买点

### 判断条件

1. 冲高回调，不是高位盘整
2. 价格落在斐波那契回撤 **0.618 / 0.786 / 0.86 / 0.94** 位置
3. 当前价格对应筹码峰的下边缘，越干净越好
4. 网格挂单跨价格线挂（保持均价围绕均线），防止故意扎针
5. **确认位置再挂，不要接刀**

### 止损原则

- 有效跌破筹码峰下边缘 → 离场
- 筹码峰整体下移 → 离场
- 不和主力对抗

### 适用范围

- 毕业（migrate 到 DEX）之后**第一个回调**胜率最高
- 市值 **$40K 以上**最佳
- 不推荐 1h 日线以上用

### 标的筛选

重点关注热门榜单和狙击榜单，每天约筛选出 **10 个**可交易标的。

### 案例

| 案例 | 斐波那契位置 | 特征 |
|------|------------|------|
| 案例一 | 61% 回撤 | 底部筹码峰买入 |
| 案例二 | 94% 回撤 | 底部筹码峰买入 |
| 案例三 | 94% 回撤 | 底部筹码峰买入 |
| 案例四 | 86% 回撤 | 底部筹码峰买入 |

## 四、交易策略 2：横盘 + 苏醒信号入场

### 原理

横盘代表"换手正在发生"，但只有"有效横盘"才代表"健康换手"。

单纯在横盘阶段买入不是高胜率策略，因为你无法判断换手完成之后是启动还是继续下移。

### 核心价值

| 阶段 | 作用 |
|------|------|
| 横盘 | 确认筹码正在换手，筹码重新分配 |
| 苏醒信号 | 确认主力已经重新入场并开始试盘 |

### 与策略 1 的区别

| 策略 1 | 策略 2 |
|--------|--------|
| 回撤后的反弹机会 | 横盘区间后拉升 |
| 无明显横盘 | 有明显横盘区间 |
| 偏后手 | 偏确认型 |

---

## 五、交易策略 3：毕业后内盘筹码换手 80% + 价格新高入场

### 原理

毕业后内盘筹码卖出 80% 以上或者价格回调 80%，本来是巨大的抛压，但是**价格不跌反而微涨**，说明做了一次彻底的主力换手。主力刚换手结束，大概率要开始拉高出货。

### 判断方法

**方法 1**: 毕业后内盘筹码卖出 80%，但价格没有跌，反而新高 → 买入

**方法 2**: 毕业后价格回撤 80%，然后反弹至新高 → 买入

### 案例

| 案例 | 类型 | 特征 |
|------|------|------|
| 案例一二三 | 直接拉升 | 内盘卖 80%，价格不跌反新高 |
| 案例四 | 回调反弹 | 内盘卖 90%，价格回调后反弹至新高 |
| 案例五 | 回调反弹 | 价格回调 80% 后反弹至新高 |
| 案例六 | 回调反弹 | 价格回调 80% 后反弹至新高 |

## 六、交易策略 4：横盘区间筹码卖出 50% 出场

### 原理

区间吸筹大概范围 10%-70%，拉升过程中主力不断出货。考虑到不是所有区间筹码都是主力筹码，以及市场波动风险，**区间筹码出货到 50% 附近基本就是本轮拉升的顶点**。

注意：50% 不一定就是顶，但 50% 处**风险大于收益，该考虑卖了**。

### 判断条件

1. 有明确的横盘区间，然后拉升
2. 圈选横盘区间，看区间净买入筹码的卖出情况
3. 卖出进度到 50% 附近 → 卖出

### 案例

四个案例均为区间卖出 50% 即为顶点。

---

## 七、交易策略 5：出货筹码峰出场

### 原理

价格高位区间，主力不断在不同价格出货，形成**高位带状的筹码峰结构**。

### 判断条件

1. 价格处于相对高位
2. 筹码峰呈现**典型带状分布**
3. 圈选高位筹码峰 > 30%
4. 辅助量价：交易量放大 + 价格上不去

### 案例

六个案例均为高位带状筹码峰，其中案例六为"假横盘、真派发"——高位价格横盘，筹码峰不断变大，横盘结束直接跳水。
