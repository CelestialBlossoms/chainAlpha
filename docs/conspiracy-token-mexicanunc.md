## 阴谋代币特征归纳

本样本表现出的“阴谋代币”特征：

1. 社交入口是 X Community 链接，而非标准项目官网，具备社区叙事传播特征。
2. CTO 标记为 `1`，更像社区接管/社区发动，而不是传统项目运营。
3. Pump.fun 毕业后迅速打出千万市值，主升段极短，注意力和资金流同时爆发。
4. Dev 历史干净，创建者不是连续发币地址，降低了“批量土狗工厂”特征。
5. Top100 净流入强，`91` 个净买钱包对 `8` 个净卖钱包，存在明显前排吸筹。
6. 前排 bundler 地址数量不高但质量高，集中在 TOP12-TOP20，且多数未卖。
7. 同时存在一组开盘附近买入、当前持续卖出的早期钱包，说明不是完全健康的单边吸筹。
8. 有 Dexscreener 广告、链接更新和 boost fee，说明传播有主动投放因素。
