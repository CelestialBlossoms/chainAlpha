## 三、金狗共性特征 (10大规律)

### 1. 极端涨幅是标配
有效金狗从K线起点到峰值涨幅 **2,400%-24,000%**。涨幅<100%的基本不是金狗。

### 2. 连续突破阳线
金狗在拉升期出现 **3-14根** 1h阳线连续突破，单时涨幅>30%的阳线出现频率很高。

### 3. 放量顶点信号
峰值处的成交量是平均成交量的 **4.5x-8.5x**。量比>5x 是见顶的强烈信号。

### 4. 短期见顶
80%的金狗在 **6-30小时内** 到达ATH。超过30h未创新高大概率已过气。

### 5. 深度回撤是常态
即使最成功的金狗，峰值后回撤 **44-86%**。没有只涨不跌的金狗。

### 6. 派发期特征
峰值后伴随量能萎缩（量趋势> -0.5）、连续阴线、反弹无力。

### 7. 积累期可识别
拉升前有 **2-6h** 的低波动积累期，价格在窄幅震荡，量能温和。

### 8. 闪拉型 vs 慢拉型
- **闪拉型** (10%): 6h内急速拉升见顶，爆发力强但持续性差
- **慢拉型** (57%): 22-30h逐步拉升，有明确的积累→突破→见顶三阶段

### 9. 19%是假金狗
约1/5进入watchlist的标的从未有效拉升，只有横盘震荡。需要通过量比和阳线比过滤。

### 10. 底部异动是先行信号
金狗在拉升前通常伴随底部筹码积累异动（Top100持仓增持、池/市值比上升）。

## 四、金狗标准生命周期 (1h K线视角)

```
┌──────────────┐    ┌──────────────────┐    ┌──────────┐    ┌────────────┐
│  积累期       │ →  │  突破期           │ →  │  顶点     │ →  │  派发期     │
│  2-6h        │    │  3-14根放量阳线    │    │  量比>5x  │    │  回撤>50%  │
│  窄幅震荡     │    │  单时涨幅>30%多次   │    │  最大量柱  │    │  量能萎缩   │
│  量能温和     │    │  阳线比>60%        │    │  ATH价格  │    │  连续阴线   │
└──────────────┘    └──────────────────┘    └──────────┘    └────────────┘
    2-6小时              主升浪                1-2小时          持续下跌
```

## 六、实时模式匹配关键指标

基于以上分析，提取以下可编程的实时匹配指标:

| 指标 | 金狗阈值 | 说明 |
|------|----------|------|
| `total_gain_pct` | >100% | 从观察起点到当前涨幅 |
| `max_hour_gain_pct` | >30% | 单时最大涨幅（突破信号） |
| `green_ratio` | >50% | 阳线占比 |
| `volume_ratio` | >3x | 最大量/均量 |
| `hours_to_peak` | <48h | 到峰值耗时 |
| `breakout_bars` | >=3 | 连续突破阳线数 |
| `drawdown_from_peak_pct` | <50% | 当前回撤（越小越好，说明还在上升期） |
| `vol_trend` | >0 | 量趋势（正数=量在增加） |
