| # | 过滤 | 条件 | 依据 |

## 失败信号画像

| 特征 | 死亡信号 | 存活信号 | 差异 |
|------|---------|---------|------|
| 价格位置(pos) | 53% | 47% | 死亡信号更倾向于高位 |
| 投降Bar出现率 | 59% | 69% | 存活信号有更多清洗 |
| 1m抢跑(最后5根全阳) | 11% | 15% | 抢跑更偏向存活 |
| 成交量趋势 | 13.6x | 10.4x | 但中位数差异不显著 |

**唯一确认的死亡模式**: "持续拉升加速型(追高风险)" — n=4, WR20=25%。信号踩在加速拉升的顶点。

**死亡信号的共性**: pos偏高 + 无投降Bar清洗 + 推送后5min跌>8%放量。

## 关键时间窗口

| 窗口 | 含义 | 决策价值 |
|------|------|---------|
| **5分钟** | 推送后第一反应 | **第一道筛选**: 涨>3%=追, 跌>8%放量=弃 |
| **10分钟** | 量能初步确认 | 1m级别前后量比>1.2x=健康 |
| **30分钟** | 趋势确认 | **第二道筛选**: 转正=加仓, 水下=减仓 |
| **1-2小时** | 主力方向明确 | 后4h走势初步成形 |
| **4小时** | 最终判决 | WR20/WR50是否达成 |

## 决策流程图

```
新代币异动 → 推送
  │
  ├─ 过滤1: 开放时间<4h? ──→ 跳过
  ├─ 过滤2: ATH<1.5x & vol<$10K? ──→ 跳过
  ├─ 过滤3: vol < $5K? ──→ 跳过
  ├─ 过滤4: K线实体<2% & 前趋势<3%? ──→ 跳过
  │
  ├─ 推送 → 附带上14种前置模式分类 + 投降Bar检测结果
  │
  └─ 推送后5min:
       ├─ 涨>3%? ──→ 🟢 强信号 (71% WR20)
       ├─ 跌>8%+放量? ──→ 🔴 危险, 降低预期
       └─ 其他 ──→ 🟡 等30min确认

推送后30min:
  ├─ 转正(>0%)? ──→ 🟢 确认, 可加仓 (68% WR20)
  └─ 仍在水下? ──→ 🟡 减仓观察 (43% WR20)
```
