## 信号类型

| 信号 | 触发条件 | 成功率 | 说明 |
|---|---|---|---|
| abnormal | 老币反弹 15%+ | 64% | 主力信号 |
| new_revival | 新币从底部启动 | 82% | 高成功率 |
| quiet_runup | 横盘后拉升 40%+ | 80% | 需确认量能 |
| quiet_breakout | 横盘后突破 10%+ | 22% | **高风险**，大市值跳过 |
| drop_40w/50w | 新币 ATH>$1M 跌破阈值 | ~50% | 抄底信号 |

## 推送过滤链

当前生效的过滤（按优先级）：

| # | 过滤 | 条件 | 效果 |
|---|---|---|---|
| 1 | 开放时间 | open_timestamp < 4h | 跳过新币（在战壕）|
| 2 | 天花板+无量 | ATH<1.5x AND vol<$10K | 成功率~0%，直接 return |
| 3 | 极度无量 | vol < $5K | 78% 失败，直接 return |
| 4 | 弱 K 线 | 信号实体<2% AND 前趋势<3% | 假突破指纹，跳过 |
| 5 | quiet_breakout 大市值 | mcap > $500K | 5/7 失败是大市值 |

## 决策流程图

```
新代币异动
  │
  ├─ open_time < 4h? ────→ 跳过（在战壕）
  │
  ├─ ATH<1.5x & vol<$10K? → 跳过（天花板+无量）
  │
  ├─ vol < $5K? ──────────→ 跳过（极度无量）
  │
  ├─ K线实体<2% & 前趋势<3%? → 跳过（弱K线）
  │
  ├─ 量能比 > 1.3x? ──────→ 🟢 推送（健康）
  │
  ├─ 量能比 < 0.5x? ──────→ 🔴 跳过（死猫跳）
  │
  └─ 0.5-1.3x ────────────→ 🟡 推送但低优先级
```
