## 三、轻度回撤 (<20%) — Bar级指纹

**概况**: n=125, WR20=66%, 是最优质的区间。

### 恢复案例的K线指纹

```
HENRY [new_revival] Peak=+3661% Trough=-8%
  [ 0min] -0.7% V=$3,892   ← 触底(几乎没跌)
  [ 5min] +268%  V=$84,033  ← 巨阳!量增20x
  → post-trough量比: 9.5x
  → 特征: 触底后5min内出现>+50%巨阳, 量增>5x

SOLMIRE [new_revival] Peak=+1039% Trough=-4%
  [ 0min] -21.8% V=$7,567
  [15min] -92.3% V=$171    ← 闪崩触底(缩量!)
  [20min] +74.6% V=$1,593  ← 第一根反弹
  [25min] +136.4% V=$2,436 ← 巨阳确认
  → 特征: 闪崩缩量 → 巨阳放量 → V反
```

### 死亡案例的K线指纹

```
CATCOIN [abnormal] Peak=+0% Trough=-20% MCap=$934K
  所有bar <±2%, 量$400-$4,500
  → 特征: 大市值, 零波动, 无人关注

PIZZA [abnormal] Peak=+3% Trough=-19%
  量$0-$379, 最大bar <±3%
  → 特征: 零量能, 资金完全不关注
```

### 区分规则

| 维度 | 恢复 | 死亡 |
|------|------|------|
| 量能 | 有>500的bar | 全部<500 |
| 最大单bar | >+5% | <±3% |
| 资金迹象 | 明显 | 完全无 |

> **轻度回撤如果量能正常(>500), 几乎是必赚的。死亡案例全都是"无量无人"型。**

## 四、中度回撤 (20-50%) — Bar级指纹

**概况**: n=129, WR20=49%, 是最常见的区间, 一半恢复一半死。

### 恢复案例的K线指纹

```
DOGE [abnormal] Peak=+193% Trough=-33%
  [ 0min] -10.9% V=$5,951
  [ 5min]  -3.9% V=$7,156   ← 缩量阴跌
  [15min] +65.6% V=$19,118  ← 巨阳!量增3x
  [30min] -13.7% V=$14,303  ← 回踩触底
  [35min] +35.4% V=$15,891  ← 二次确认
  → post-trough量比: 1.3x
  → 特征: 触底后出现>20%放量阳线

CJP [abnormal] Peak=+225% Trough=-30%
  [0-35min] 缓慢缩量阴跌到底
  → post-trough量比: 2.1x (量能翻倍!)
  → 绿/红: 11/5 (绿多)
  → 特征: 触底后量增+绿多, 稳步回升
```

### 死亡案例的K线指纹

```
Wcup [new_revival] Peak=+7% Trough=-50%
  [ 5min] +1.5% V=$489   ← 峰值(连10%都不到)
  [20min] -11.7% V=$339
  [30min] -1.6% V=$12     ← 量崩塌!
  → 最大阳线仅+9%, 量萎缩到$10
  → 特征: 反弹无力, 量能枯竭

SACKS [abnormal] Peak=+6% Trough=-50%
  [15min] +9.1% V=$940    ← 小反弹
  [25min] -20.6% V=$4,612 ← 二次暴跌! 量>反弹量
  → 特征: 反弹后出现更大的阴线
```

### 区分规则

| 维度 | 恢复 | 死亡 |
|------|------|------|
| 最大单bar涨幅 | **>20%** | <15% |
| 触底后量比 | **>1.5x** | <1.0x |
| 绿红比 | 绿>红 | 红>绿 |
| 二次暴跌 | 无 | 常有 |

> **中度回撤的生命线: 触底后能否出现一根>20%的放量阳线。能→大概率恢复。不能→大概率死亡。**

## 五、重度回撤 (50-80%) — Bar级指纹

**概况**: n=97, WR20=41%, 64%死亡。BABEL所在的区间。

### 恢复案例的K线指纹 (41%)

```
AOC [abnormal] Peak=+456% Trough=-56%
  [ 0min] -20.1% V=$9,912
  [25min] -52.9% V=$1,172  ← 缩量触底(量只有开头的1/9!)
  [30min] +21.7% V=$1,790  ← 第一根反弹阳
  [35min] +42.0% V=$2,112  ← 继续放量
  [40min] +143.5% V=$15,714 ← 爆发!量增13x!
  → post-trough量比: 4.6x (量能爆发)
  → 最大单bar: +144%
  → 特征: 缩量到底 → 逐步放量反弹 → 巨阳爆发

Gena [new_revival] Peak=+100% Trough=-78%
  [10min] -76.7% V=$5,116  ← 暴跌触底
  [15-35min] 缩量横盘洗盘
  [45min] +79.4% V=$4,180  ← 爆发!
  [50min] +71.7% V=$3,870
  [55min] +49.8% V=$11,959 ← 天量确认
  → 特征: 暴跌→缩量横盘→放量爆发
```

### 死亡案例的K线指纹 (59%)

```
Aliens [abnormal] Peak=+8% Trough=-80%
  [ 0min] -2.9% V=$9,542   ← 推送即巅峰
  [ 5min] -21.2% V=$6,713
  [10min] -13.1% V=$17,155  ← 放量暴跌
  [20min] -28.0% V=$13,995
  [35min] -72.2% V=$6,309  ← 加速下跌
  → post-trough量比: 0.1x (量崩塌!)
  → 特征: 持续放量下跌, 触底后量能枯竭, 无有效反弹

mk4 [abnormal] Peak=+10% Trough=-78%
  [ 0min] +1.5% V=$2,483
  [30min] +1.4% V=$3,081  ← 弱反弹
  [35min] -69.8% V=$16,338 ← 巨阴!二次崩盘
  → 特征: 弱反弹后巨阴杀跌
```

### 区分规则

| 维度 | 恢复 | 死亡 |
|------|------|------|
| 触底时量能 | **缩量**(卖压枯竭) | 放量(恐慌持续) |
| 触底后量比 | **>3x**(爆发) | <1x(枯竭) |
| 最大单bar涨幅 | **>40%** | <25% |
| 触底后走势 | 逐步放量→巨阳爆发 | 持续阴跌或二次巨阴 |
| 15min内确认 | **有**巨阳 | 无 |

> **重度回撤生命线: 触底前必须缩量(卖压枯竭), 触底后15min内出现>20%放量阳线。满足→试错,止损-15%。不满足→放弃。36%WR20不是0%, 关键看实时K线。**

## 六、极端回撤 (>80%) — Bar级指纹

**概况**: n=12, WR20=25%, **WR50=0%**。跌超80%的没有一个能翻倍。

```
AFFIRM [new_revival] Peak=+44% Trough=-92%
  [ 0min] -19.2%
  [ 5min] +62.7% V=$22,961 ← 峰值(短暂反弹)
  [30min] -75.6% V=$13,128 ← 再次崩盘
  [35min] -84.2%           ← 继续跌
  → 即使"恢复"案例, 实际也只短暂弹到+44%然后继续跌

INCOME [new_revival] Peak=+0% Trough=-91%
  [ 0min] -15.0%
  [10min] -42.3% V=$12,679 ← 持续放量暴跌
  [20min] +19.1% V=$12,446 ← 短暂反弹
  [35min] -42.1%           ← 继续跌
  → 反弹无力, 放量阴线主导
```

### 区分规则

| 维度 | 恢复 | 死亡 |
|------|------|------|
| WR50 | **0%**(全区间) | 0% |
| 最大反弹 | <+63% | <+20% |
| 量能 | 反弹放量但后继无力 | 阴线持续放量 |

> **极端回撤: WR20仅25%, WR50=0%。不建议入场。跌超80%说明基本面已崩。**

## 七、通用入场决策树

```
推送后CA出现回撤
  │
  ├─ 回撤<20% (轻度)
  │   ├─ 有量(>500) → 5min涨>3%直接入场 (WR20=66%, new_revival=92%)
  │   └─ 无量(<500) → 放弃(归零率极高)
  │
  ├─ 回撤20-50% (中度)
  │   ├─ 触底后出现>20%放量阳线(量>前均1.5x) → 入场,止损-10%
  │   └─ 最大阳线<15% → 放弃(大概率死亡)
  │
  ├─ 回撤50-80% (重度)
  │   ├─ 触底时缩量 + 触底15min内出现>20%放量阳线(量增>2x) → 试错,止损-15%
  │   └─ 触底时放量 或 15min内无巨阳 → 放弃(64%死亡)
  │
  └─ 回撤>80% (极端)
      └─ 放弃(WR50=0%, 不值得)
```

## 八、量能速查表

| 回撤区间 | 恢复组触底后量比 | 死亡组触底后量比 | 恢复组最大单bar | 死亡组最大单bar |
|---------|---------------|---------------|---------------|---------------|
| 轻度 | 9.5x | 1.0x | +268% | +1% |
| 中度 | 1.3~2.1x | 0.5~0.8x | +66% | +16% |
| 重度 | 4.6x | 0.1~0.5x | +144% | +24% |
| 极端 | 0.1~0.2x | 0.2x | +63% | +19% |

**核心规律: 恢复组的触底后量比总是>1.3x, 死亡组总是<1.0x。量能是区分生死的最强单一指标。**

## 九、一句话总结

```
轻度回撤+new_revival     → 闭眼入(92%WR20, 0%归零)
中度回撤+放量阳>20%      → 可入场
重度回撤+缩量触底+巨阳   → 试错, 止损-15%
重度回撤+放量触底+无巨阳 → 放弃
极端回撤                 → 永远不碰
```
