# 深 V vs 死猫跳 识别指南

## 概述

底部异动推送后，代币通常会先回撤再拉升（深 V），也可能反弹即巅峰（死猫跳）。本文档总结区分这两者的量能识别方法。

## 三步判断法

### 第一步：拉 K 线，找到信号时刻

推送时间是 14:30，拉 14:30 前后的 5m K 线：

```
14:15  vol=$500
14:20  vol=$800
14:25  vol=$600  ← 信号前 3 根，算平均 = $633
14:30  vol=$2,000 ← 信号 K 线（放量拉升）
14:35  vol=$1,200 ← 信号后第 1 根
14:40  vol=$900  ← 信号后第 2 根
14:45  vol=$700  ← 信号后第 3 根，算平均 = $933
```

### 第二步：计算量能比

```
信号前 3 根均量: $633
信号后 3 根均量: $933

量能比 = 后/前 = 933/633 = 1.47x
```

### 第三步：判定

| 量能比 | 判定 | 操作建议 |
|---|---|---|
| **> 1.3x（放量）** | 🟢 Healthy V | 有人在持续买，V 反可信，可加仓 |
| **0.7x ~ 1.3x（持平）** | 🟡 正常 | 量没崩，继续观察，可轻仓 |
| **< 0.5x（缩量）** | 🔴 Dead Cat Bounce | 拉完没人接了，清仓/不跟 |

## 实战案例

### 案例 1：死猫跳（$ALgoat）

```
信号前 3 根均量：$101
信号后 3 根均量：$45

量能比 = 0.45x → 缩量 55%
→ 🔴 死猫跳，信号后没人接了
→ 结果：+3% 短暂反弹后暴跌 -68%
```

### 案例 2：健康 V 反（$ket）

```
信号前 3 根均量：$1,059
信号后 3 根均量：$2,163

量能比 = 2.04x → 放量 104%
→ 🟢 健康 V 反，有持续大单跟进
→ 结果：+63%
```

### 案例 3：缩量但奇迹拉回（Weak V，$Mira）

```
信号前 3 根均量：$492
信号后 3 根均量：$212

量能比 = 0.43x → 缩量
→ 但最终涨了 +109%
→ 罕见案例，不建议赌
```

## 数据支撑

基于 2026-05-18 29 个深 V/DCB 代币的统计分析：

| 量能趋势 | 成功 | 失败 | 成功率 |
|---|---|---|---|
| UP（放量 >1.3x）| 9 | 3 | **75%** |
| FLAT（0.7-1.3x）| 8 | 6 | 57% |
| DOWN（缩量 <0.5x）| 3 | 3 | 50% |

双确认（量能 DOWN + ATH<1.5x）：成功率接近 0%。

## 失败代币的其他特征（辅助判断）

除了量能，以下的特征也高度关联失败：

| 特征 | 失败率 | 说明 |
|---|---|---|
| ATH/当前 < 1.5x（天花板）| 52% | 没有拉升空间 |
| 市值 > $500K | 47% | 拉升成本高 |
| 信号 K 线实体 < 2% | 78% | 假阳线 |
| 信号前 5 根涨幅 < 3% | 68% | 没有上涨动能 |
| 开放时间 < 4h（在战壕）| 67% | 新币不稳定 |
| 下午 12:00-20:00 推送 | 40% | 垃圾时段 |

## 最佳实践

1. **推送后不要立刻跟**——等 3 根 5m K 线（15 分钟），看到量能确认再行动
2. **量能 DOWN 的直接放弃**——哪怕看起来跌了很多"抄底"，大概率是接飞刀
3. **量能 UP + ATH>1.5x + 市值 $50-200K** = 最优信号
4. **5m 线判断时效，15m 线确认趋势**——两者都放量最可靠
5. **下午推送的要更谨慎**——下午时段成功率比凌晨低 25%
