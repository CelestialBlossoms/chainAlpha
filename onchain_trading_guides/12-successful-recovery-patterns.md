## 五、四条共性规律

### Pre推送前

| 共性 | 详情 |
|------|------|
| **必须有横盘/企稳** | 4个全部有: 暴跌后反弹企稳(HENRY/Bank) 或 缩量横盘(hypu/Mira) |
| **推送在企稳后** | 不是在暴跌中推送、不是在反弹尖上推送、是在横盘末端或反弹初期推送 |
| **量能特征** | 横盘期成交量持续萎缩(缩量=卖压枯竭) |
| **价格位置** | 地板价pos<20%(HENRY/Bank) 或 中位横盘pos 30-50%(hypu/Mira) |

### Post推送后

| 共性 | 详情 |
|------|------|
| **先跌后涨或直接涨** | 不会直接暴跌。回踩幅度<10%(hypu dip -7.4%) |
| **第一波拉升** | 在推送后5-35min内出现, 涨幅>10% |
| **量能爆发** | 拉升时量>横盘期3-20x |
| **到顶时间** | 25-200min, 不是5分钟瞬爆, 有入场窗口 |

### 对比: 成功 vs 失败

| 维度 | 成功组 | 失败组 |
|------|--------|--------|
| Pre横盘 | ✅ 均有横盘/企稳 | ❌ 无横盘 |
| Push位置 | 底部(pos<20%)或中位横盘末端 | 反弹尖上或持续下跌中 |
| 量能 | 横盘期缩量 | 无缩量过程 |
| PoolRatio | ≤40% | ≥34%(边缘值) |

## 六、入场决策规则

```
1. Pre必须有横盘/企稳阶段(至少6根bar缩量窄幅)
2. Push在横盘末端/反弹初期(不在暴跌中, 不在反弹尖上)
3. Post回撤<10% → 入场; 回撤10-20% → 轻仓; 回撤>20% → 放弃
4. 优选地板价(pos<20%), 其次中位横盘(pos 30-50%)
5. PoolRatio>40%降低仓位
```
