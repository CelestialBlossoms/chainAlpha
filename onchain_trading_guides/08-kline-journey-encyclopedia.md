## 二、new_revival — 7种K线走势详解

### 走势1：底部持续下跌（占比21%，WR20=84%，最优结构）

**前置K线画像：** 价格在48根K线范围的下30%，Q4正在加速下跌(-35%)，波动率极高。

```
Q1(+2.4%) → Q2(-0.3%) → Q3(-13.4%) → Q4(-34.7%)  → 信号触发
 ↑横盘       ↑开始跌      ↑加速跌      ↑暴跌        ↑在最低点附近触发
```

**推送后4h分叉：**

| 后4h走势 | 概率 | WR20% | WR50% | MedPeak | 盈亏特征 |
|----------|------|-------|-------|---------|---------|
| 暴涨 | **59%** | **100%** | **100%** | **+130%** | 先涨24%→回调7%→横盘→再涨8%。持仓体验：先甜后苦再甜 |
| 温和上涨 | 16% | 80% | 40% | +31% | 先跌15%→反弹5%→回调8%→涨7%。持仓体验：先苦后甜 |
| 暴涨回吐 | 12% | 100% | 0% | +29% | 先跌5%→暴跌36%→再跌27%→反弹8%。持仓体验：过山车，到过峰但没守住 |
| 持续阴跌 | 12% | 0% | 0% | +16% | 一路向东南，-20%→-25%→-23%→-1%。**唯一亏损分支** |

**策略：** 59%概率暴涨，等Q4跌完+后1-2h回调(-7%)入场。12%阴跌风险通过后4h走势可完全识别。

---

### 走势2：持续拉升中（占比18%，WR20=74%，分化最大）

**前置K线画像：** 价格持续走高，Q3/Q4加速拉升，已从底部涨了不少。

```
Q1(-0.5%) → Q2(+10.8%) → Q3(+35.3%) → Q4(+23.4%)  → 信号触发
 ↑横盘       ↑启动拉升    ↑加速        ↑继续拉      ↑火上浇油
```

**推送后4h分叉：**

| 后4h走势 | 概率 | WR20% | MedPeak | 盈亏特征 |
|----------|------|-------|---------|---------|
| 暴涨 | 33% | **100%** | +99% | 继续飞，先涨18%→微调5%→横盘→微跌6% |
| 冲高急跌 | 30% | **100%** | +42% | 先跌8%→续跌15%→再跌8%→微跌8%。到过峰但急速回落 |
| 持续阴跌 | **30%** | **12%** | +5% | 先跌18%→续跌4%→再跌27%→跌10%。**最差分支** |
| 暴涨回吐 | 4% | 100% | +150% | 先跌9%→暴跌21%→反弹14%→跌35% |

**策略：** 三等分——1/3继续飞，1/3冲高急跌（设止盈能赚钱），1/3阴跌死。**关键过滤器：0-1h如果跌超10%，立刻放弃（100%不会是暴涨分支）。**

```

**推送后4h分叉：**

## 八、实战操作流程

```
Step 1: 推送发生，识别信号类型（new_revival/abnormal）

Step 2: 判断前置结构（参考第二节、第三节）
  new_revival:
    底部持续下跌/横盘 → 高价值，重点跟踪
    持续拉升中 → 中性，需后4h确认
    底部反弹启动 → 谨慎，39%会阴跌
  abnormal:
    底部反弹启动 → 最优(基线80%)
    持续拉升中 → 占49%但需筛选，只有30%值得做

Step 3: 观察后4h走势（关键决策点，参考第四节）
  暴涨/冲高急跌/强涨守住 → 入场（WR20=100%）
  温和上涨 → new_revival可轻仓，abnormal放弃
  持续阴跌 → 坚决放弃

Step 4: 确认量能（参考第七节）
  平量/放量+暴涨 → 正常
  缩量+任何 → 谨慎

Step 5: 执行入场
  底部结构+暴涨：1-2h回调时入场
  拉升结构+暴涨：0-1h确认涨就追
  高点回落+暴涨：企稳后入场

Step 6: 设止盈止损
  50%仓位 +30%止盈
  30%仓位 +50%止盈
  20%仓位 +100%止盈
  止损：-25%硬止损
```

```
