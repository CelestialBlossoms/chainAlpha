## 七、止盈策略

```
分批限价止盈：
  50%仓位 → +30% 止盈
  30%仓位 → +50% 止盈
  20%仓位 → +100% 止盈

止损：-25% 硬止损

为什么不用固定时间出场？
  后4h"暴涨"信号的中位峰值+113%，但持4h只能拿到+10%
  → 限价止盈才能锁住利润，时间出场必然回吐
```

## 八、完整操作流程

```
Step 1: 推送发生 → 识别信号类型（new_revival 优先）

Step 2: 判断前置K线结构（参考第六节）
  底部持续下跌/横盘 → 高价值，重点跟踪
  持续拉升中 → 中性，等后4h确认
  底部反弹启动 → 谨慎，看后4h是否阴跌

Step 3: 观察后4h走势（关键决策点，参考第五节）
  暴涨/冲高急跌/强涨守住 → ✅ 入场
  持续阴跌 → ❌ 放弃

Step 4: 入场（参考第四节）
  new_revival: 底部结构→等1-2h回调入；拉升结构→0-1h确认涨就追
  abnormal: 等1-4h，确认后4h不阴跌再入

Step 5: 执行止盈止损（参考第七节）
```

## 九、不碰清单

| 条件 | 理由 |
|------|------|
| 后4h持续阴跌 | 胜率20%=20-26% |
| quiet_runup | 推送时已涨+93%，追高必死 |
| 死拿不做止盈 | 峰值+100%，时间出场盈亏仅+1~10% |
| abnormal <50K + 不是底部反弹启动 | 胜率20%仅50% |
| 等8h再入场（new_revival） | 峰值中位90min，8h后行情早结束了 |
