# Bar级K线交易策略

> 基于363个信号的逐根5m+1m K线指纹分析, 全本地计算
> 术语见: [08-K线指纹百科全书](./08-5m-fingerprint-encyclopedia.md)

---

## 一、推送后的第一个5分钟就已经告诉你答案了

**这是本策略最核心的发现**: 推送后5分钟内K线的走向, 就能预测最终结果。

| 推送后5min走势 | 信号数 | 其中WR20 | 解读 |
|--------------|--------|---------|------|
| **涨>3%** | 70个 | **71%** | 推送后立刻被抢筹, 大概率是个好信号 |
| 横盘(-3%~+3%) | 88个 | 47% | 方向不明, 需要再观察 |
| 跌>3% | 132个 | 45% | 先跌后涨模型, 不一定差但需要等 |

> 推送后5分钟就涨超过3%的, 71%最终能到+20%。**5分钟涨=可以直接追。**

---

### 推送后30分钟: 更准的确认信号

| 推送后30min状态 | 信号数 | WR20 |
|---------------|--------|------|
| **已经转正(>0%)** | 115个 | **68%** |
| 还在水下(<0%) | 155个 | 43% |

> 30分钟已经从跌变涨了 → 68%最终能到+20%。30分钟还趴在水下 → 只有43%。

## 二、单根Bar决定命运: 投降Bar是最强预测信号

### 什么是投降Bar？

**5分钟内暴跌超8% + 成交量是平时的3倍以上**

大白话: 有人扛不住了, 不计成本砸盘跑路。这一砸把想卖的人都清空了, 后面反而容易涨。

### 数据说话

| 推送前30min内 | 信号数 | WR20 | 平均峰值 |
|-------------|--------|------|---------|
| 有投降Bar | 48个 | **66.7%** | **+139.2%** |
| 没有投降Bar | 131个 | 45.0% | +42.4% |

> 有投降Bar的信号, 平均能冲到+139%。没有的只有+42%。**这是363个信号中最强的单一预测因子, 没有之一。**

### 投降Bar之后的三种结局

投降Bar本身不决定一切, 关键看投降之后K线怎么走:

```
投降Bar发生后:
  │
  ├─ 后面出现锤子线(长下影)/放量阳线 + 价格反弹
  │   → "恐慌投降→V反恢复型" (134个, WR20=53.7%)
  │   → 最常见, 占37%的信号
  │
  ├─ 后面继续阴跌, 没看到任何承接信号
  │   → "恐慌投降→持续下跌型" (59个, WR20=59.3%)
  │   → 虽然名字叫持续下跌, 但其中不少后面V回来了(AvgPeak=+122%)
  │
  └─ 后面缩量横盘, 没人买卖了
      → "恐慌投降→横盘筑底型" (39个, WR20=56.4%)
      → WR50最高的一类(35.9%)
```

## 三、价格位置决定赔率: 地板价 vs 天花板价

| 推送时价格位置(pos) | 信号数 | WR20 | 平均峰值 |
|-------------------|--------|------|---------|
| **地板价(pos<20%)** | 63个 | **66.7%** | **+113.8%** |
| 中低位(20-40%) | ~80个 | ~55% | ~+50% |
| 中位(40-60%) | ~80个 | ~50% | ~+40% |
| 中高位(60-80%) | ~80个 | ~48% | ~+35% |
| **天花板价(pos>80%)** | 54个 | **44.4%** | **+40.4%** |

> 地板价(pos<20%) = 抄在大底部, WR20高达66.7%, 平均峰值+114%。
> 天花板价(pos>80%) = 已经涨过了才推送, WR20只有44.4%。

## 四、1m级别抢跑信号

推送前最后5根1m K线全是阳线 → 有人抢跑, 在推送前就偷偷买了。

| 1m最后5根 | 信号数 | WR20 | 平均峰值 |
|----------|--------|------|---------|
| **全阳(抢跑)** | 48个 | **60.4%** | +45.5% |
| 其他 | 315个 | ~50% | ~+55% |

> 抢跑信号WR20=60.4%, 比平均水平(51.8%)高9个百分点。

## 五、唯一确认的死亡模式

**持续拉升加速型(追高风险)**: n=4, WR20=**25.0%**

特征: 推送前连续多段持续上涨, 且成交量越放越大(越涨越追)。推送恰好发生在拉升最高点。

大白话: 信号踩在拉升的顶点上 — 该买的人都买完了, 后面没有新买家, 只能跌。

## 六、实战操作流程

### 推送时立刻看3个东西:

**1. 有没有投降Bar?** (最近30min内)
- 有 → +20%概率=66.7%, 重点跟踪
- 没有 → 普通信号, 放低预期

**2. 价格在哪里?** (pos)
- pos<20%(地板价) → 好信号, 大概率弹
- pos 20-60% → 中等, 等确认
- pos>80%(天花板价) → 谨慎, 可能已经涨完了

**3. 投降Bar后面怎么走的?**
- 有锤子线/放量阳线 → 可入场("V反恢复型")
- 还在跌无承接 → 再等等
- 缩量横盘 → 等放量突破再入

### 推送后等5-10分钟:

- **5min涨>3%** → 可以直接追(71%概率到+20%)
- **5min跌<3% + 量正常** → 等30min确认
- **5min跌>8% + 放量** → 恐慌未结束, 放弃

### 30分钟后再看:

- **30min已经转正** → 加仓(68%概率), 最佳确认窗口
- **30min还在跌** → 减仓或等4h大级别确认

### 仓位管理:

| 条件 | 仓位 | 止盈 | 止损 |
|------|------|------|------|
| 投降Bar + pos<20% + 5min涨 | 正常仓 | +30%/+50%/+100%分批 | -25% |
| 投降Bar + 其他 | 半仓 | +30%/+50% | -20% |
| 无投降Bar + pos>80% | 不参与或极小仓 | +20% | -15% |

## 七、一句话总结

```
有投降Bar + 地板价 + 推送后5分钟涨 → 闭眼入, 大概率大肉
无投降Bar + 天花板价               → 放弃, 基本是陷阱
30分钟转正                        → 确认信号, 可以加仓
```
